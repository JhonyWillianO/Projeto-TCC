para poder upar aqui no github 
devemos começar ir na pasta do projeto e ir diretorio e colocar "CMD"
logo em seguida dar um git add .
logo depois git commit -m "frase que quiser aqui"
depois fazer git branch -M main e depois usar git remote add origin link da sua repositorio do github
e dar por ultimo git push -u origin main
