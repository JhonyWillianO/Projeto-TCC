---------- Tutorial Para Abrir Projeto ---------------
1- abaixar game maker e estalar ele na maquina https://gamemaker.io/pt-BR/download.
2- extrair arquivo qualquer pasta.
3- após estalar GameMaker é clicar no icone do GameMaker com nome "jogoDetiro" e abrir projeto, descrição é "GameMaker Project".
4- para testar o projeto clicar somente icone play(executar) ou usar tecla F5.
